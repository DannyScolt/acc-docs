## ADDED Requirements

### Requirement: Receivables opening balance clarification
Items 2 and 3 in section 2.2.10 (cong no khach hang and cong no nha cung cap dau ky) SHALL include a clarification note in the description column stating that Phase 1 only captures opening balances, and the full receivables/payables management module (reconciliation, payment tracking, aging) is in Phase 2-3.

#### Scenario: Accountant reads receivables opening balance
- **WHEN** an accountant reads item 2 (cong no khach hang dau ky) or item 3 (cong no nha cung cap dau ky) in section 2.2.10
- **THEN** the description includes text clarifying that full accounts receivable/payable management is available in Phase 2-3

### Requirement: Inventory opening balance clarification
Item 4 in section 2.2.10 (ton kho dau ky) SHALL include a clarification note stating that Phase 1 only captures opening inventory balances, and the full warehouse management module (receipts, issues, transfers, stocktake) is in Phase 2.

#### Scenario: Accountant reads inventory opening balance
- **WHEN** an accountant reads item 4 (ton kho dau ky) in section 2.2.10
- **THEN** the description includes text clarifying that full warehouse operations are available in Phase 2

### Requirement: Fixed assets and tools opening balance clarification
Items 7, 8, and 9 in section 2.2.10 (TSCD, CCDC, CPTT dau ky) SHALL include a clarification note stating that Phase 1 captures aggregate opening balances for the corresponding accounts, and the detailed management modules (depreciation, amortization, transfer, disposal) are in Phase 6.

#### Scenario: Accountant reads fixed assets opening balance
- **WHEN** an accountant reads item 7 (TSCD dau ky), item 8 (CCDC dau ky), or item 9 (CPTT dau ky) in section 2.2.10
- **THEN** the description includes text clarifying that detailed asset/tool/prepaid management is available in Phase 6
